# Importing Data

To use AmazonasDataHub’s datasets, make sure you already have the
package installed (see Tutorial - Install). Para usar as bases de dados
do AmazonasDataHub, primeiramente, garanta que o pacote já esteja
instalado (veja a aba de Tutorial - Instalação).

Carregue o pacote usando a função `library()`: Import the package using
`library()`

```r
library(amazonasdatahub)
```

## AmazonasDataHub - Datasets

Until now, the `amazonasdatahub` package includes four datasets for you,
wheter you are a student, statistician, or enthusiast, to analyze and
apply statistical methods.

Até o momento, o pacote `amazonasdatahub` conta com quatro bases de
dados para você, estudante, estatístico ou entusiasta, para analisar e
aplicar métodos estatísticos.

-   base_manaus: Time series with Extreme Values of Limited Range;
-   malaria_amazonas: An integrated dataset of malaria notifications in
    the state of Amazonas;
-   pib_trimestral: Gross Domestic Product - GDP of Amazonas;
-   rionegro_amazonas: A dataset of the Rio Negro River (Amazonas) level
    from 2003 to 2023.

As soon as the package is loaded, these datasets can be accessed.

Assim que o pacote é carregado, essas bases de dados já podem ser
acessadas.

### Documentation

Each dataset of `amazonasdatahub` is documented, explaining details
about the dataset and what each variable represents. To view the
documentation, simply use `?` followed by the dataset name.

Cada base de dados do `amazonasdatahub` é documentada, explicando
detalhes sobre a base de dados e o que cada variável representa. Para
ver a documentação, basta utilizar o `?` precedido do nome da base de
dados.

```r
?base_manaus
?malaria_amazonas
?pib_trimestral
?rionegro_amazonas
```

`rionegro_amazonas`, which contains data on the water level of the Negro
River from 2003 to 2023.

dados do nível do Rio Negro, de 2003 a 2023.
