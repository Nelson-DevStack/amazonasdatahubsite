# Instalação

Você pode instalar o `amazonasdatahub` utilizando o `devtools` do R. Se
você não possuir o `devtools` instalado, instale-o usando:

```r
install.packages("devtools")
```

Após ter instalado o `devtools`, você pode instalar o `amazonadatahub`:

```r
library(devtools)
devtools::install_github("Nelson-DevStack/amazonasdatahub")
```
