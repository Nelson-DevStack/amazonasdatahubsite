# Tutorial - Início

Vamos instalar e usar o AmazonasDataHub.

## Início

### O que você vai precisar:

Para instalar o `amazonasdatahub`, é necessário que seu você possua
instalado no seu computador ou ambiente de desenvolvimento:

-   R version 4.41.1 (2024-06-14) – “Race for Your Life”;
-   Pacote `devtools` do R.

## Instalando o devtools

Se você não possuir o `devtools` instalado, instale-o usando no terminal
do R:

```r
install.packages("devtools")
```

## Instalando o amazonasdatahub

Após ter instalado o `devtools`, ainda no terminal do R, você pode
instalar o pacote `amazonadatahub` através do método `install_github()`:

```r
library(devtools)
install_github("Nelson-DevStack/amazonasdatahub")
```

## Usando no R

Assim que o `amazonasdatahub` for instalado, use `library()` ou
`require()` para carregar o pacote.

```r
library(amazonasdatahub)
```
